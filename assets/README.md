# Иконки и Splash Screen

Положи сюда два файла перед запуском `npm run mobile:icons`:

| Файл | Размер | Назначение |
|------|--------|------------|
| `icon.png` | 1024×1024 | Иконка приложения (без скруглений, PNG) |
| `splash.png` | 2732×2732 | Сплэш-скрин (логотип по центру) |

Команда `npx capacitor-assets generate` автоматически создаст все
нужные размеры для iOS и Android из этих двух файлов.

## Рекомендации по дизайну

- **Иконка**: зелёный фон `#16a34a`, белая корзина по центру
- **Splash**: белый/светлый фон `#fafaf9`, логотип «Ценомер» по центру
- Формат: PNG, без прозрачности для splash

## Быстрая генерация через Figma / Canva

1. Фон 1024×1024, цвет `#16a34a`
2. Иконка корзины (ShoppingBasket) белая, ~512×512, по центру
3. Экспорт в PNG
