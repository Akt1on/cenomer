# Ценомер — лучший агрегатор цен на продукты в Москве

## Финальная профессиональная версия (июнь 2026)

Проект полностью оптимизирован для бесплатного использования:
- Инкрементальный скрапинг + режимы quick/full
- Автоматический ежедневный запуск скрапера
- Оптимизация Supabase (остаёмся в 500 МБ)
- Полная автоматизация через GitHub Actions
- Улучшенный Dependabot

### Как запустить локально
```bash
cd scraper
python main.py --dry-run --mode quick
```

### Автоматический запуск
Скрапер запускается автоматически каждый день в 06:00 по Москве (через GitHub Actions).

**Необходимо настроить секреты в Settings → Secrets and variables → Actions:**
- SUPABASE_URL
- SUPABASE_SERVICE_ROLE_KEY
- TELEGRAM_BOT_TOKEN (опционально)
- TELEGRAM_CHAT_ID (опционально)
- ANTHROPIC_API_KEY (опционально, для AI-категоризации)

См. SUPABASE_OPTIMIZATION.md для деталей по базе данных.
